# arch_hyprland_dotfiles

## Components

- **Login Manager:** `ly`  
- **Compositor:** `hyprland`  
- **Status Bar:** `waybar`  
- **Wallpaper Manager:** `hyprpaper`  
- **Lock Screen:** `hyprlock`  
- **Shell:** `fish`  
- **Terminal:** `kitty`  
- **Editors:** `nvim`, `vim`  
- **File Manager:** `dolphin`  
- **System Info:** `fastfetch`  
- **Notifications:** `dunst`
